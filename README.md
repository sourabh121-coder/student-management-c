# Student Management System

A simple **Student Management System** built in C to manage student records efficiently. This project allows users to perform CRUD (Create, Read, Update, Delete) operations on student data such as ID, name, marks, and more. It is designed for educational purposes and beginner programmers to understand file handling, structures, and basic database management in C.

## Features

- Add new student records
- View all student records
- Search for a student by ID
- Update student information
- Delete student records
- Simple console-based interface
- Data stored in files for persistence

## Technologies Used

- **Programming Language:** C
- **IDE/Editor:** Visual Studio Code or any C IDE
- **File Handling:** Used for storing student data

## Project Structure

StudentManagementSystem/
│
├── main.c # Main program file
├── student.c # Functions for student operations
├── student.h # Header file with structures and function declarations
├── data.txt # File to store student data
└── README.md # Project documentation


## Installation & Running

1. **Clone the repository:**
```bash
git clone https://github.com/your-username/StudentManagementSystem.git
```

##2. Navigate to the project directory:
```bash
cd StudentManagementSystem
```
##3. Compile the C program:
```bash
gcc main.c student.c -o student_management
```
##4. Run the program:
```bash
./student_management
```
